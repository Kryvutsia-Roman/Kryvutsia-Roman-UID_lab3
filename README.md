# Kryvutsia-Roman-UID_lab3

Десктопний застосунок «Форма опитування», розроблений для лабораторної роботи №3 з курсу «Розробка інтерфейсів користувача».

##  Опис проєкту
Застосунок реалізує інтерфейс для проведення послідовного опитування користувача. Програма відображає запитання, отримує відповідь через текстове поле та зберігає результати у локальний файл.

##  Технологічний стек
* **Мова програмування:** Python 3.x
* **GUI Фреймворк:** PySide6 (Qt for Python)
* **Стилізація:** Qt Style Sheets (QSS)
* **Збереження даних:** Стандартна бібліотека os та робота з файлами (UTF-8)
